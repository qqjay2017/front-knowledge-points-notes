## 如何实现圣杯布局和双飞翼布局


圣杯布局/双飞翼布局
1. 使用float布局
2. 两侧使用margin负值,以便和中间内容横向重叠
3. 防止中间内容被两侧覆盖,一个用padding,一个用margin


### 圣杯布局



```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .content {
            padding-left: 150px;
            padding-right: 200px;
        }
        .content .column {
            float: left;
        }
        .middle {
            width: 100%;
            background-color: blue;
        }
        .left {
            position: relative;
            left: -150px;
            width: 150px;
            margin-left: -100%;
            background-color: red;
        }
        .right {
            width: 200px;
            margin-right: -100%;
            background-color: aqua;
        }

        .footer {
            clear: both;
        }

    </style>
</head>
<body>
    <header>header</header>
    <div class="content">
        <div class="middle column">中间的</div>
        <div class="left column">left</div>
        <div class="right column">right</div>
    </div>
    <footer class="footer">footer</footer>
</body>
</html>
```


### 双飞翼布局

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <style>
      .column {
        float: left;
      }
      .main {
        margin-left: 100px;
        margin-right: 200px;
      }
      .middle {
        width: 100%;
      }
      .left {
        width: 100px;
        margin-left: -100%;
        background-color: aqua;
      }
      .right {
        width: 200px;
        margin-left: -200px;
        /* margin-left: -100%; */
        background-color: aquamarine;
      }
    </style>
  </head>
  <body>
    <div class="middle column">
      <div class="main">mainmainmainmainmainmain</div>
    </div>
    <div class="left column">left</div>
    <div class="right column">right</div>
  </body>
</html>


```
