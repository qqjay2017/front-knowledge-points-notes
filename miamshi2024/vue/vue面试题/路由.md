## hash的特点

hash变化会触发网页的跳转,即浏览器的前进后退
hash变化不会刷新页面
hash不会提交到serve端



### 修改hash

```
location.href = '#/home'
```


### 获取hash

```
location.hash
```


### 监听hash变化

```
window.onhashchange = function(event){

}

```
