## 认知 webAPI

### 学习目标
* 理解
  * 能说出API 是什么
  * 能说出webAPI 是什么
  * 能够说出webAPI 由哪两个部分组成


### 1. API 是什么

**概念:** API（Application Programming Interface,应用程序编程接口）`是一些预先定义的函数`，目的是提供应用程序与开发人员基于某软件或硬件得以访问一组例程的能力，而又无需访问源码，或理解内部工作机制的细节。

- 任何开发语言都有自己的API
- API的特征输入和输出(I/O)
  - var max =  Math.max(1, 2, 3);
- 如何学习API: 
  - 掌握这个API的功能(作用)
  - 掌握这个API要传递的参数
  - 掌握这个API执行完毕之后返回值是什么

### 2.webAPI 是什么

**概念:**  浏览器提供的一套操作浏览器功能(BOM)和页面元素的API(DOM)

我们学习webAPI主要学习两大部分

- DOM 
- BOM
